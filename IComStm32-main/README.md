# IComStm32

An STM32-based project featuring C++ SPI abstraction and communication logic.

## Project Structure
- `Inc/`: Header files
- `Src/`: Source files
- `Drivers/`: STM32 HAL/CMSIS drivers (CubeMX managed)

## Development
This project is configured for **STM32CubeIDE** and maintains compatibility with **STM32CubeMX**.
